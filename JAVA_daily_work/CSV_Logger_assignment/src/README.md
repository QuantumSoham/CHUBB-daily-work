# WeeklyAssignmentStreams

Eclipse project demonstrating:
- Java Streams & Lambdas for report generation (CSV)
- CompletableFuture (async) API simulation
- Log parser using streams & lambdas
- Refactor of imperative parts to functional style

## How to run
1. Import this folder as an *Existing Projects into Workspace* in Eclipse, or just compile in your IDE.
2. Run `com.soham.weekly.Main` with Java 11+.

## Files
- src/com/soham/weekly/*.java : source code
- resources/sales.csv : sample CSV dataset
- resources/logs.txt : sample logs
